Blockchain Ticketing System

A Web3 DApp on Ethereum Sepolia Testnet

Welcome to the Blockchain Ticketing System

This decentralized application (DApp) demonstrates a simple ticketing system using blockchain technology. The system is built on the Ethereum Sepolia Testnet and implements ERC-20 token standard for tickets.

Available Features

Create a new wallet for the Ethereum blockchain or load an existing one.

Decrypt Wallet

Decrypt your existing wallet using a keystore file and password.

Go to Wallet Decryption
